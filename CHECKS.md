# 對客頁驗收清單（2026-07-20 立——簽核頁七輪返工的學費）

任何對客頁（簽名／分享／表單／工具頁）改動，交付前逐項過。
**交付訊息必附三欄**：哪些格實測過／哪些格靠設計保證／哪些格需要 Wayne 真機掃一眼。籠統的「測過了」不算交付。

## 環境矩陣（逐格）

| 環境 | 重點 |
|---|---|
| LINE 內建瀏覽器 iOS／Android | 腳本會不會死、file input 沒有拍照、簽名板 |
| 外部瀏覽器 手機 Safari／Chrome | 版面貼合、觸控 |
| 桌機 Chrome **已登入任意 Google 帳號** | 內嵌內容白框（帳號轉跳擋 iframe） |
| 桌機 無痕 | 基準對照 |
| **手機捲動中** | 網址列收合觸發 resize——版面定案後必須上鎖 |
| 載入過程（不只完成畫面） | 資源陸續載入造成的高度跳動、按鈕位移 |

## 硬規則（違反即擋）

1. **對客頁禁止 iframe／資源直連 script.google.com**——資料走 fmt=json、文件走 srcdoc 同源內嵌；登入 Google 的瀏覽器會把 GAS 內嵌擋成白框
2. **依視窗尺寸計算的版面：量定後上鎖**——resize 事件只准在定案前作用
3. **版本核對**：交付連結帶新 `?v=N`、頁腳版字自動同步網址版號、驗收先確認拿到的是新版
4. **改版前先讀該頁「歷史定案」**——遷移／改版必須把舊驗收準則當清單帶著走
5. **位置語意先對齊**——「預覽」「按鈕」這類詞先確認指哪一頁哪一區再動手
6. 症狀第二次出現＝停手換架構、不准再貼補丁

## 各頁歷史定案

### quote-sign（簽核頁、2026-07-20 Wayne 驗收定版）
標題 → **整份報價單同源內嵌**（同底色淡影、無多層框、整頁攤開跟頁面同捲、高度上鎖、文件內「複製帳號」可點）→ **下載 PDF 一顆** → 聲明 → 簽名。
無資訊卡（金額在文件裡）、無放大鈕、公司抬頭統編只出現一次（簽章區）、頁腳只留地址電話。
訂閱式：文件內月費（含稅）主打、首次支付明細行、押金不計稅。

### 2026-07-21 柏軒實案（P0）
- **JSON 注入 `<script>` 必跳脫 `<`**（`JSON.stringify(x).replace(/</g,"\\u003c")`）：資料含 `</script` 字樣會提前關閉標籤、整頁 script 死、剩餘內容變亂碼——凡 server inject JSON 進頁面一律套用
- 頁面注入不夾大型 HTML（docHtml 類走 fetch 通道）；驗收必含「未簽回件」狀態
